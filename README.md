# Gang Shalik Restaurant Website

Welcome to the GitHub repository for Gang Shalik, a serene oasis where nature meets culinary delight. Our website is designed to showcase the enchanting ambiance of Gang Shalik, nestled amidst lush greenery, complete with tranquil water features and charming cottages. It's a vibrant space designed for families, featuring a dedicated kids play zone and captivating bird sanctuary.

## Features

- **Home Page**: A warm welcome to visitors with an overview of what Gang Shalik has to offer.
- **About Page**: Dive into the story of Gang Shalik, our philosophy, and what makes us unique.
- **Menus Page**: Explore our diverse menu, featuring a wide range of dishes that cater to various tastes and preferences.
- **Gallery Page**: Take a virtual tour through our gallery and immerse yourself in the enchanting ambiance of Gangshalik.
- **Contact Page**: Get in touch with us for reservations, events, or inquiries. Includes a contact form for easy communication.

## Technology Stack

This project is built using the following technologies:

- **Next.js**: A React framework for building server-side rendering and static web applications.
- **Tailwind CSS**: A utility-first CSS framework for rapidly building custom designs.
- **React Hook Form**: Efficient, flexible, and extensible forms with easy-to-use validation.

## Getting Started

To get a local copy up and running follow these steps:

1. **Prerequisites:**
    - Ensure you have Node.js (version 14 or later) and npm (Node Package Manager) installed on your system. You can download them from the official website [Node.js download](https://nodejs.org/en/download/).

2. **Clone the repository:**
    - Open your terminal and navigate to the directory where you want to clone the repository.
    - Run the following command to clone this repository:

      ```bash
      git clone [https://github.com/your-username/gang-shalik-restaurant-website.git](https://github.com/your-username/gang-shalik-restaurant-website.git)
      ```

      Replace `your-username` with your actual GitHub username.

3. **Install dependencies:**
    - Navigate to the project directory:

      ```bash
      cd gang-shalik-restaurant-website
      ```

    - Run the following command to install the project's dependencies:

      ```bash
      npm install
      ```

4. **Run the development server:**
    - Start the development server using the following command:

      ```bash
      npm run dev
      ```

    - This will start the server and open the website in your default browser, typically at http://localhost:3000/.

5. **Development:**
    - Edit the website code in the `src` directory.
    - The development server will automatically recompile and update the browser whenever you make changes to the code.
